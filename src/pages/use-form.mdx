# Use Form

React integration for the [@corets/form](https://github.com/corets/form) package.

```bash
yarn add @corets/use-form
```

## `useForm`

Use form inside a React component and re-render whenever something changes:

```tsx
import React from "react"
import { createForm } from "@corets/form"
import { useForm } from "@corets/use-form"

const Example = () => {
  const form = useForm(() => createForm({ field: "foo" }))

  return (
    <input value={form.get("field")} onChange={(e) => form.set("field", e.target.value)} />
  )
}
```

**Note:** Read more about forms in [@corets/form](/form) docs.
