# Value

Simple observable value that can also be used inside React using Hooks. This observable is accompanied by [@corets/list](/list) and [@corets/store](/store).

```bash
yarn add @corets/value
```

## React

You can use values inside React using this hooks based integration [@corets/use-value](/use-value).

## LocalStorage

There is also a version of this library that syncs its state to the localStorage: [@corets/local-storage-value](/local-storage-value)

## `createValue`

Creates a new instance of `Value`:

```ts
import { createValue } from "@corets/value"

const count = createValue(0)
```

Create a new instance without the factory function:

```ts
import { Value } from "@corets/value"

const value = new Value(0)
```

Create a new instance with a custom differ:

```ts
import { createValue } from "@corets/value"

const differ = (oldValue, newValue) => true

const count = createValue(0, { differ })
```

## `Value.get`

Retrieve value:

```ts
value.get()
```

## `Value.set`

Replace value:

```ts
value.set(2)
```

## `Value.listen`

Listen to any changes:

```ts
value.listen((value) => console.log(value))
```

You can also choose to invoke the listener immediately like this:

```ts
value.listen((value) => console.log(value), { immediate: true })
```

You can provide a custom differ too:

```ts
const differ = (oldValue, newValue) => true

value.listen((value) => console.log(value), { differ })
```

## `Value.use`

Returns value, and a setter function to update the underlying value. This is a convenience method for people used to React's `useState` syntax.

```ts
import { createValue } from "@corets/value"

const [value, setValue] = createValue(1).use()
```
